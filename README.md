Hello) It's my little project 'Contacts'

All you need to do is follow the steps:

1. install dependencies in /app and /server directories

 ```yarn```
 or
 ```npm install```

 2. go to /server dir and run:

 ```yarn start```
 or
 ```npm start```

 3. go to /app dir and run:

   ```yarn start```
   or
   ```npm start```
